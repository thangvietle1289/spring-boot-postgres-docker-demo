# The application for spring boot + postgresdb + docker

## 1. Build project
`$ mvn clean install -DskipTests`

## 2. Go to docker folder
`$ cd src/main/docker/`

## 3. Run docker-compose
`$ docker-compose up`

## 4. Test
Navigation to [http://localhost:8080](http://localhost:8080) and show product list
