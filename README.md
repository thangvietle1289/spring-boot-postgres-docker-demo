# The application for spring boot + postgresdb + docker

## Images
1. [Openjdk image](https://hub.docker.com/_/openjdk/)
2. [Postgres docker image](https://store.docker.com/images/postgres)

## Build maven project
`$ mvn clean install -DskipTests`

## Go to docker folder
`$ cd src/main/docker/`

## Run docker compose
`$ docker-compose up`
